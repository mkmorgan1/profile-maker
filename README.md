# profile-maker

A social media application

## Installation

```bash
  npm install
```

Ensure MongoDB is installed and running

## Run

Node

```bash
  npm start
```

PM2

```bash
  npm run startPm2
    or
  npm run stopPm2
```

Development (nodemon)

```bash
  npm run dev
```

## Test

```bash
  npm test
```

## Webpack

```bash
  npm run build
```
